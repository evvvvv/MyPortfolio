## 一块——基于事件的校园社交平台
「一块」是本科时和同学共同完成的一个创业项目，是一个发布／接受委托的校园半数人事件社交平台，所有委托的报酬皆为一块钱。旨在以委托作为载体，挖掘委托中的社交属性，打造一个安全健康的校园社交平台。
<br>
在项目中我负责前期需求分析、竞品分析和原型设计，并独立完成app所有页面的UI设计。
